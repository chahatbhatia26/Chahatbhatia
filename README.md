# Chahat Bhatia Portfolio 2026

Portfolio website for Chahat Bhatia, Product Designer.

## Run Locally

From this folder:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://127.0.0.1:8000/
```

## Main Pages

- `index.html` — portfolio homepage
- `case-study.html` — professional work case study
- `exploration-case-study.html` — personal exploration case study
- `guided-investing-case-study.html` — guided investing personal project case study

## Deploy Later

This is a static site, so it can be deployed on GitHub Pages, Netlify, Vercel, or any static hosting provider.
